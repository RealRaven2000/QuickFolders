Menu On Top: History

Have a look at the quick tutorial here: How to install Menu On Top

Menu On Top is a small extension which makes it possible to style Thunderbird and Firefox main menu. Detail on how to use it and screenshots are here.

Version Details

Here is a history of changes - as always note the dates are in UK format DD/MM/YYYY...

  • Version 1.16

    • Updated color picker controls for Thunderbird 68.
    • Updated all numeral textboxes for Thunderbird 68.
    • Added Swedish / Italian translations for 'Fire-orange' flavour.
    • Experimental: Added the "Rise of Tools" functionality (partly). This will elevate the main toolbar above the tab bar. however the toolbar is not hidden on calendar tabs, so this is kind of unfinished work.

    Note: I still plan to add a routine that automatically unpacks all avatar graphics automatically - at the moment you will have to unpack them from the Add-on manually.

  • Version 1.15 - 17/06/2019

    • Improved loading UI on Thunderbird 68 beta.
    • Support for new HTML color picker which replaces the XUL version of same in modern versions of Thunderbird.
    • Support for new Preferences system, but backwards compatible with Thunderbird 60.
    • Replacement of document.createElement with createXULElement
    • Improved loading custom icon (avatar) in modern versions of Thunderbird.

  • Version 1.14.1 - 07/05/2019

    • Made compatible with Thunderbird beta 67.0. The minimum application for this version is Thunderbird 60. Works on Waterfox as well.
    • Replaced all colorpickers with HTML elements (standard color picker, OS dependant). Unfortunately, these only support hexadecimal values; transparency and alpha values for colors are currently not possible.
    • Replaced all groupbox elements with HTML fieldsets
    • Added advanced debug settings which can be configured through right-clicking the debug checkbox on MenuOnTop options - Advanced Tab.
    • 1.14.1 - removed javascript console warnings caused by MenuOnTop unnecessarily running on dialog windows.

  • Version 1.13 - 21/02/2019

    • Special Edition for the Interlink Mail client
    • Fixes the wrong position of the Menu in Interlink. (was shown underneath main tooplbar, now on top).
      Note that the options "Show Tabs in Title" is currently unsupported.

  • Version 1.11 - 19/02/2019

    • Made compatible with Thunderbird 66.0b1 and current version of Pale Moon.
    • Added new darker orange color choice 'fire'.
    • Improved positioning of status bar icon when it is configured - navigation bar is only used as a last resort, the configuration button tries to install itself into the status bar if it finds one.

  • Version 1.10 - 15/10/2018

    I am planning on making compatible versions of my other Add-ons QuickFolders quickFilters, and SmartTemplates as well but this will take some time - I am planning a developer hangout where I am hoping to get some help converting one of these as an example for existing developer of legacy Add-ons.   • Version 1.9 - 27/05/2018

    • Added more color flavors (May Green, Cloud Blue, Rose Pink, Photon Bright & Dark)

      Current Purely Color Based Options:

      • Cloud - a light blue
      • Mountain - deep blue, ideal for styling Thunderbird
      • Parakeet - exotic bluish green, similar to Jade

      • Lime - sweet light green

      • Tangerine - deep orange matched with Firefox

    • Added menu border style options (solid, hidden, inset, outset, groove, ridge, dotted, dashed, double)
    • Changed Radius settings from em to px because Thunderbird removed the em binding in XUL input items.
    • Added new Avatar Icon for Thunderbird 60.
    • Added function to sanitise the CSS for menu backgrounds (removes background: and semicolon when pasting rules from gradient editor at colorzilla.com)

  • Version 1.7 - 26/11/2017

    Important news: This addon will not work from Firefox version 57 as XUL / XPCOM based addons are officially not supported anymore. To keep using MenuOnTop you can install Waterfox (recommended), Firefox Pale Moon or SeaMonkey. These are all browsers based on the Mozilla open & free codebase. To install, maked sure to enable extensions.legacy.enabled = true in about:config.
    Some more information what changed in Firefox Quantum and how to get your Addons back:
    • Removed obsolete references to nsIPrefBranch2. These would break when reading settings in Thunderbird 57.
    • Removed version options from script tags which also broke settings in Thunderbird 57 and later.
    • Fixed storage of some default settings (top margin, left margin)

  • 1.0 - 11/Oct/2015

    • Added lop-left bookmark feature, use it to store your most important web pages, single messages (Tb) and mail folders (Tb). This is a handy and quick way of keeping only the most important bookmarks within easy reach without wasting screen space.

Navigate to any web-page or open an email, open a single message tab or click on a Folder, and then choose "Add current Item" (Thunderbird) or "Add current Webpage" (Firefox).
