Release 6.3 Please note that QuickFolders is compatible with the latest version of Thunderbird (115.3.2) - if you get a message that says otherwise after updating from earlier versions (e.g. Tb91) or if QuickFolders cannot be enabled with the blue slider, simply remove it and reinstall it via "Find more Add-ons" search box on the Extensions tab of Add-ons Manager.
